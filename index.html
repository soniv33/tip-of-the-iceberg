More operations

The rest of what a real Iceberg table can do.

🗑️ Delete (copy-on-write vs merge-on-read)

Copy-on-write rewrites files without the rows. Merge-on-read drops a small "skip this row" note and hides them at read time (compaction folds it away later).

✏️ Update / Upsert

Update sets a column for matching rows. Upsert merges a row by title (replace-or-insert = MERGE).

🧹 Compact · Expire · Evolve

Compact merges small files & folds in delete notes. Expire keeps only the latest snapshot and deletes unused files. Add column evolves the schema (old rows read null).
